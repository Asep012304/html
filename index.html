HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk mendeskripsikan struktur dan isi dari sebuah halaman web. HTML bukan bahasa pemrograman, melainkan bahasa markup yang menggunakan elemen-elemen tertentu untuk menyusun konten.


Dengan HTML, Kamu dapat membangun fondasi dari sebuah halaman web, yang kemudian dapat diperindah dengan CSS (Cascading Style Sheets) dan ditambah interaktivitas menggunakan JavaScript.
